# 💰 Finance Analysis Dashboard

## 📌 Project Overview

The **Finance Analysis Dashboard** is an interactive business intelligence project developed using **Microsoft Power BI**. It provides comprehensive insights into financial transactions, tax collection, customer segments, transaction status, and state-wise performance.

The raw data was cleaned and prepared using **Microsoft Excel** before being visualized in Power BI.

---

## 🎯 Project Objectives

- Analyze financial transaction data.
- Monitor total amount, transaction count, and average transaction value.
- Track tax collection across different dimensions.
- Analyze customer segments and transaction status.
- Identify state-wise financial performance.
- Provide interactive business insights for better decision-making.

---

## 🛠️ Tools Used

- **Microsoft Excel** – Data Cleaning & Preparation
- **Microsoft Power BI** – Dashboard Design & Data Visualization

---

## 🧹 Data Cleaning

The dataset was cleaned using Microsoft Excel by:

- Removing duplicate records
- Handling missing values
- Standardizing column names
- Correcting data formats
- Validating transaction records
- Preparing data for Power BI analysis
